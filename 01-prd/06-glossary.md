# 06. 용어 사전 (Glossary)

> **문서 목적**: 프로젝트 전체에서 사용되는 핵심 용어, 엔티티 상태값, 화면 번호 매핑을 통일하여 Phase 2(UX 플로우), Phase 3(UI 와이어프레임) 등 후속 작업의 모든 에이전트가 동일한 언어로 소통할 수 있도록 한다.
> **대상 독자**: 기획자, UX 디자이너, 개발자, 모든 에이전트 (Phase 2, 3 필수 참조)
> **중요도**: 이 문서는 모든 후속 산출물에서 반드시 참조해야 하는 기준 문서이다.

## 1. 핵심 용어 매핑

| 한국어 | 영문 (엔티티/코드명) | 정의 |
|--------|---------------------|------|
| 순찰 | Patrol | GPS 기반 트래킹 활동. 사용자가 앱에서 시작 버튼을 눌러 경로를 기록하고, 종료 시 거리/시간/포인트가 산출되는 단위 활동. |
| 마킹 / 현장 기록 | PatrolMark | 순찰 중 특정 위치에서 사진과 메모를 남기는 기록. 지도 위에 핀으로 표시된다. |
| 신고 | Report | 순찰 중 이상 상황(불법투기, 시설파손, 안전위험 등)을 보고하는 행위. 마킹의 하위 유형(type='report')으로 생성된다. |
| 대원 / 회원 | User | 앱에 가입한 사용자. 반려견주, 방범대원, 일반 산책러를 포괄한다. |
| 관리자 | User (role=admin) | 관리자 웹에 접근하여 대원 관리, 통계 조회, 이벤트/포인트/알림 관리를 수행하는 사용자. |
| 피드 / 게시판 | Post (Community Feed) | 커뮤니티 탭의 게시글 목록. 활동인증/정보공유/자유 3개 카테고리로 구분된다. |
| 포인트 | Points | 순찰 거리에 비례하여 자동 지급되는 앱 내 보상 단위. 기본 공식: 1km = 100pt. |
| 랭킹 / 리더보드 | Ranking (Leaderboard) | 사용자들의 순찰 활동(거리 기준)을 집계하여 순위를 매긴 목록. 실시간/주간/월간으로 구분. |
| 이벤트 | Event | 관리자가 등록하는 기간 한정 활동. 보너스 포인트 배율을 설정할 수 있다. |
| 푸시 알림 | PushNotification | 관리자가 발송하는 모바일 알림. 전체 또는 특정 그룹 대상. |
| 경로 | Route (GeoJSON) | 순찰 중 수집된 GPS 좌표의 연결선. 지도 위에 폴리라인으로 표시된다. |
| 활동 일지 | Patrol.note | 순찰 종료 시 사용자가 입력하는 소감/특이사항 텍스트. |
| 어뷰징 | Abuse Detection | 비정상적 속도(차량 이동, GPS 조작)로 부정하게 포인트를 획득하려는 행위. |
| 보너스 배율 | Event.bonus_multiplier | 이벤트 기간 동안 포인트를 기본 공식 대비 N배로 지급하는 설정값. |
| 반려견 | Dog | 사용자가 앱에 등록하는 반려견 엔티티. 이름·견종·성별·생년월일·중성화·예방접종 정보를 포함한다. |
| 견종 | breed | 반려견의 품종. 목록에서 선택하거나 직접 입력할 수 있다. |
| 중성화 | neutered | 반려견의 중성화 수술 여부. Dog.is_neutered 필드로 관리된다. |
| 예방접종 | vaccinated | 반려견의 예방접종 완료 여부. Dog.is_vaccinated 필드로 관리된다. |

## 2. 엔티티 상태값 정의

### 2.1 Patrol.status — 순찰 상태

| 상태 | 값 | 설명 | 진입 조건 | 전이 가능 |
|------|----|------|-----------|-----------|
| 대기 | `ready` | 순찰 레코드가 생성됨, 아직 시작 전 | 순찰 시작 버튼 탭 직전 | → `active` |
| 진행 중 | `active` | GPS 트래킹이 활성화되어 위치를 수집 중 | 시작 버튼 탭 또는 재개 버튼 탭 | → `paused`, → `completed` |
| 일시정지 | `paused` | GPS 수집이 일시 중단됨, 타이머 정지 | 일시정지 버튼 탭 | → `active` (재개), → `completed` (종료) |
| 완료 | `completed` | 순찰이 종료되어 거리/시간/포인트가 확정됨 | 종료 버튼 탭 또는 30분 정지 시 자동 | (최종 상태) |

```
상태 흐름:
ready → active ⇄ paused → completed
                 active → completed
```

### 2.2 Report.status — 신고 처리 상태

| 상태 | 값 | 설명 | 진입 조건 | 전이 가능 |
|------|----|------|-----------|-----------|
| 접수됨 | `submitted` | 사용자가 신고를 제출한 초기 상태 | 사용자가 U06에서 신고 제출 | → `in_progress` |
| 처리 중 | `in_progress` | 관리자가 신고를 확인하고 조치 진행 | 관리자가 A03에서 상태 변경 | → `resolved` |
| 처리 완료 | `resolved` | 신고 건이 최종 처리됨 | 관리자가 A03에서 처리 완료 | (최종 상태) |

```
상태 흐름:
submitted → in_progress → resolved
```

### 2.3 User.role — 사용자 역할

| 역할 | 값 | 설명 | 접근 범위 |
|------|----|------|-----------|
| 일반 사용자 | `user` | 앱의 일반 기능 이용 | 모바일 앱: U00~U25 |
| 관리자 | `admin` | 대원 관리, 통계, 운영 기능 이용 | 모바일 앱 + 관리자 웹: A01~A10 |

### 2.4 PatrolMark.type — 마킹 유형

| 유형 | 값 | 설명 |
|------|----|------|
| 일반 기록 | `normal` | 일반적인 현장 사진/메모 기록 |
| 신고 | `report` | 이상 상황 신고. Report 엔티티가 함께 생성됨 |

### 2.5 Post.board_type — 게시판 유형

| 유형 | 값 | 설명 |
|------|----|------|
| 활동인증 | `activity` | 순찰/산책 활동을 인증하는 게시글 |
| 정보공유 | `info` | 지역 안전 정보, 유용한 팁 공유 |
| 자유 | `free` | 주제 제한 없는 자유 게시글 |

### 2.6 Ranking.period_type — 랭킹 기간

| 유형 | 값 | 설명 | 갱신 주기 |
|------|----|------|-----------|
| 일간 | `daily` | 당일 누적 기준 | 순찰 종료 시 즉시 |
| 주간 | `weekly` | 해당 주(월~일) 누적 기준 | 매일 자정 |
| 월간 | `monthly` | 해당 월 누적 기준 | 매일 자정 |

### 2.7 PushNotification.target_type — 알림 대상

| 유형 | 값 | 설명 |
|------|----|------|
| 전체 | `all` | 모든 사용자에게 발송 |
| 그룹 | `group` | 특정 조건의 사용자 그룹에게 발송 |

## 4. 약어 및 기술 용어

| 약어/용어 | 설명 |
|-----------|------|
| GPS | Global Positioning System, 위성 기반 위치 측위 시스템 |
| GeoJSON | 지리 데이터를 JSON 형식으로 표현하는 국제 표준 |
| FCM | Firebase Cloud Messaging, Google의 Push 알림 서비스 |
| APNs | Apple Push Notification service, Apple의 Push 알림 서비스 |
| CRUD | Create, Read, Update, Delete — 기본 데이터 조작 |
| MVP | Minimum Viable Product, 최소 기능 제품 |
| P0/P1/P2 | 기능 우선순위 등급 (P0: 필수, P1: 중요, P2: 희망) |
| CSV | Comma-Separated Values, 표 형식 데이터 파일 |
| FK | Foreign Key, 외래 키 (엔티티 관계 참조) |
| UUID | Universally Unique Identifier, 범용 고유 식별자 |
| 딥링크 | Deep Link, 앱 내 특정 화면으로 직접 이동하는 링크 |
