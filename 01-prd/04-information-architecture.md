# 04. 정보 구조 (Information Architecture)

> **문서 목적**: 앱과 관리자 웹의 전체 네비게이션 구조를 정의하고, 각 화면에 고유 번호를 부여하여 기획/디자인/개발 간 커뮤니케이션 기준을 제공한다.
> **대상 독자**: UX 디자이너, 기획자, 개발자
> **참조**: 화면 번호 ↔ 기능 매핑은 `06-glossary.md` 참조

---

## 1. 모바일 앱 네비게이션 트리

```
앱
│
├── 온보딩
│   ├── U00  스플래시
│   ├── U00a 온보딩 슬라이드 (3장)
│   ├── U00b 로그인
│   ├── U00c 회원가입
│   └── U00d 비밀번호 찾기
│
├── [탭 1] 홈 (순찰)
│   ├── U02  순찰 대기 (지도 + 시작 버튼)
│   │   ├── U02-empty  반려견 미등록 시 빈 상태 → U24 (반려견 등록 유도)
│   │   └── U25  산책 일지 (서브탭: 월별 캘린더 + 반려견 통계)
│   ├── U03  순찰 진행 중 (지도 + 경로 + 타이머 + 속도)
│   ├── U04  순찰 일시정지 (정지/재개/종료 버튼)
│   ├── U05  현장 마킹 (사진 + 메모 + 유형 선택)
│   ├── U06  신고 작성 (카테고리 + 설명 + 사진)
│   ├── U07  순찰 결과 (거리/시간/포인트 요약 + 일지 입력)
│   └── U08  순찰 기록 상세 (과거 순찰 경로 + 마킹 목록)
│
├── [탭 2] 커뮤니티
│   ├── U10  피드 (카테고리 탭: 활동인증/정보공유/자유)
│   ├── U11  게시글 상세 (본문 + 이미지 + 댓글)
│   ├── U12  게시글 작성 (제목 + 본문 + 이미지 + 카테고리)
│   └── U13  영상 목록 / 재생 (썸네일 + 인앱 플레이어)
│
├── [탭 3] 랭킹
│   └── U14  리더보드 (실시간/주간/월간 탭)
│
├── [탭 4] 마이페이지
│   ├── U15  프로필 (프로필 이미지 + 닉네임 + 총 거리/포인트)
│   │   └── U24  반려견 관리 (반려견 등록/수정 폼)
│   │       └── U24a 견종 선택 (검색 + 목록 + 직접 입력)
│   ├── U16  활동 내역 (순찰 이력 목록 → U08 상세)
│   ├── U17  순찰 기록 목록 (날짜별 순찰 리스트)
│   ├── U18  포인트 내역 (획득/차감 이력)
│   ├── U19  프로필 편집 (닉네임 + 프로필 이미지 변경)
│   └── U20  설정 (알림 설정 + 로그아웃 + 탈퇴 + 앱 정보)
│
├── 공통
│   ├── U09  알림 목록 (헤더 벨 아이콘에서 진입)
│   ├── U21  위치 권한 요청 팝업
│   ├── U22  카메라/갤러리 권한 요청 팝업
│   └── U23  에러/네트워크 오류 화면
│
└── 하단 탭바
    ├── 홈 (순찰)
    ├── 커뮤니티
    ├── 랭킹
    └── 마이페이지
```

## 2. 관리자 웹 네비게이션 트리

```
관리자 웹
│
├── A01  대시보드
│        ├── 총 대원 수
│        ├── 금일/금주 순찰 횟수
│        ├── 누적 순찰 거리
│        └── 활성 순찰 수 (현재 진행 중)
│
├── A02  대원 관리
│        ├── 대원 목록 (검색/필터/정렬)
│        └── 대원 상세 (순찰 이력 + 마킹 이력 + 포인트 이력)
│
├── A03  신고 관리
│        ├── 신고 목록 (상태별 필터)
│        └── 신고 상세 (사진 + 지도 + 처리 이력)
│
├── A04  이벤트 관리
│        ├── 이벤트 목록
│        └── 이벤트 등록/수정 (제목 + 설명 + 기간 + 보너스 배율)
│
├── A05  포인트 관리
│        ├── 포인트 수동 지급/차감
│        └── 포인트 이력 조회
│
├── A06  Push 알림 관리
│        ├── 알림 작성/발송
│        └── 발송 이력
│
├── A07  통계
│        ├── 기간별 순찰 통계 (일/주/월)
│        ├── 대원별 활동 순위
│        ├── 지역별 순찰 밀도 (히트맵)
│        └── CSV 다운로드
│
├── A08  콘텐츠 관리
│        ├── 영상 URL 관리 (CRUD)
│        └── 게시판 카테고리 관리
│
├── A09  로그인
│
└── A10  설정
         ├── 관리자 계정 관리
         ├── 포인트 정책 설정 (기본 배율, 일일 상한)
         └── 어뷰징 임계값 설정
```

## 3. 화면 번호 매핑 테이블

### 3.1 모바일 앱 화면 (U00~U25)

| 번호 | 화면명 | 탭/섹션 | 주요 기능 |
|------|--------|---------|-----------|
| U00 | 스플래시 | 온보딩 | 앱 로고 표시 + 자동 전환 |
| U00a | 온보딩 슬라이드 | 온보딩 | 앱 소개 (3장 스와이프) |
| U00b | 로그인 | 온보딩 | 이메일/비밀번호 로그인, 소셜 로그인 |
| U00c | 회원가입 | 온보딩 | 이메일, 비밀번호, 닉네임, 휴대폰 번호 |
| U00d | 비밀번호 찾기 | 온보딩 | 이메일 입력 → 재설정 링크 발송 |
| U02 | 순찰 대기 | 홈 | 현재 위치 지도 + 순찰 시작 버튼 |
| U03 | 순찰 진행 중 | 홈 | 실시간 경로 + 타이머 + 거리 + 속도 |
| U04 | 순찰 일시정지 | 홈 | 정지/재개/종료 버튼 |
| U05 | 현장 마킹 | 홈 | 사진 촬영 + 메모 + 유형(일반/신고) 선택 |
| U06 | 신고 작성 | 홈 | 카테고리 + 설명 + 사진 |
| U07 | 순찰 결과 | 홈 | 거리/시간/포인트 요약 + 활동 일지 입력 |
| U08 | 순찰 기록 상세 | 홈 | 과거 순찰 경로 지도 + 마킹 목록 |
| U09 | 알림 목록 | 공통 | 수신 알림 목록 (읽음/미읽음) |
| U10 | 피드 | 커뮤니티 | 게시글 목록 (카테고리 탭) |
| U11 | 게시글 상세 | 커뮤니티 | 본문 + 이미지 + 댓글 + 좋아요 |
| U12 | 게시글 작성 | 커뮤니티 | 제목 + 본문 + 이미지 + 카테고리 선택 |
| U13 | 영상 목록/재생 | 커뮤니티 | 영상 썸네일 목록 + 인앱 재생 |
| U14 | 랭킹 (리더보드) | 랭킹 | 실시간/주간/월간 탭 + 순위 목록 |
| U15 | 프로필 | 마이페이지 | 프로필 이미지 + 닉네임 + 총 통계 |
| U16 | 활동 내역 | 마이페이지 | 순찰 이력 목록 |
| U17 | 순찰 기록 목록 | 마이페이지 | 날짜별 순찰 리스트 |
| U18 | 포인트 내역 | 마이페이지 | 포인트 획득/차감 이력 |
| U19 | 프로필 편집 | 마이페이지 | 닉네임, 프로필 이미지 변경 |
| U20 | 설정 | 마이페이지 | 알림 설정, 로그아웃, 탈퇴, 앱 정보 |
| U21 | 위치 권한 요청 | 공통 | GPS 권한 요청 팝업 |
| U22 | 카메라/갤러리 권한 | 공통 | 카메라/갤러리 접근 권한 요청 팝업 |
| U23 | 에러 화면 | 공통 | 네트워크 오류, 서버 오류 안내 |
| U24 | 반려견 등록/수정 | 반려견 | 사진·이름·견종·성별·생년월일·중성화·예방접종·성격 등록/수정 폼 |
| U24a | 견종 선택 | 반려견 | 검색 + 목록에서 견종 선택 또는 직접 입력 |
| U25 | 산책 일지 | 순찰 | 월별 캘린더 + 반려견 통계 카드(총 횟수/거리/시간/상위 %) |

### 3.2 관리자 웹 화면 (A01~A10)

| 번호 | 화면명 | 주요 기능 |
|------|--------|-----------|
| A01 | 대시보드 | 핵심 지표 요약 (대원 수, 순찰 현황, 신고 현황) |
| A02 | 대원 관리 | 대원 목록/상세/검색/필터 + CSV 추출 |
| A03 | 신고 관리 | 신고 목록/상세/상태 변경 + CSV 추출 |
| A04 | 이벤트 관리 | 이벤트 CRUD + 보너스 배율 설정 |
| A05 | 포인트 관리 | 수동 지급/차감 + 이력 조회 |
| A06 | Push 알림 관리 | 알림 작성/발송 + 발송 이력 |
| A07 | 통계 | 기간별/대원별/지역별 통계 + CSV 다운로드 |
| A08 | 콘텐츠 관리 | 영상 URL CRUD + 게시판 카테고리 관리 |
| A09 | 로그인 | 관리자 이메일/비밀번호 로그인 |
| A10 | 설정 | 관리자 계정, 포인트 정책, 어뷰징 임계값 설정 |

## 4. 화면 전환 흐름 요약

### 4.1 순찰 흐름
```
U02 (순찰 대기)
 → U03 (순찰 진행 중)
    → U04 (일시정지) → U03 (재개) 또는 U07 (종료)
    → U05 (현장 마킹)
       → U06 (신고 작성, 유형='신고' 선택 시)
    → U07 (순찰 결과)
       → U08 (순찰 기록 상세)
```

### 4.2 커뮤니티 흐름
```
U10 (피드)
 → U11 (게시글 상세) → 댓글 작성/좋아요
 → U12 (게시글 작성) → 저장 → U10
 → U13 (영상 목록) → 영상 재생
```

### 4.3 마이페이지 흐름
```
U15 (프로필)
 → U16 (활동 내역) → U08 (순찰 기록 상세)
 → U17 (순찰 기록 목록) → U08
 → U18 (포인트 내역)
 → U19 (프로필 편집)
 → U20 (설정)
```

### 4.4 온보딩 흐름
```
U00 (스플래시)
 → U00a (온보딩 슬라이드, 최초 실행 시)
 → U00b (로그인)
    → U00c (회원가입)
    → U00d (비밀번호 찾기)
 → U02 (홈, 로그인 완료 후)
```
