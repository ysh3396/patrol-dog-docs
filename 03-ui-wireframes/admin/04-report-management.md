### 인터랙션 설명 (상세 패널)

| 요소 | 동작 | 결과 |
|------|------|------|
| [X] 닫기 | 클릭 | 사이드 패널 닫기, 목록 화면 복귀 |
| 첨부 사진 클릭 | 이미지 확대 | 라이트박스 모달로 원본 사진 확대 표시 |
| 지도 영역 | 신고 위치(lat, lng) 핀 표시 | 지도 확대/축소 가능 |
| 처리 상태 드롭다운 | 상태 전이 선택 | 아래 전이 규칙 참조 |
| 처리 메모 입력 | 텍스트 입력 (admin_note) | 상태 변경 시 함께 저장 |
| [저장] 버튼 클릭 | 상태 변경 + 메모 저장 | 확인 팝업 → 저장 완료 토스트, 목록 갱신 |

### Report.status 전이 규칙

| 현재 상태 | 변경 가능 상태 | 추가 동작 |
|-----------|---------------|-----------|
| `submitted` | `in_progress` | 관리자 메모(admin_note) 입력 (선택) |
| `in_progress` | `resolved` | 처리 결과 메모 입력 (선택), resolved_at 기록, 신고자에게 PushNotification 발송 옵션 |
| `resolved` | (변경 불가) | 최종 상태 |
