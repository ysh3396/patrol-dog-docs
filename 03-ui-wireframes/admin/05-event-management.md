### 이벤트 상태 표시 규칙

| 상태 | 조건 | 뱃지 색상 |
|------|------|----------|
| 진행중 | 시작일 <= 오늘 <= 종료일 | 초록 |
| 예정 | 오늘 < 시작일 | 파랑 |
| 종료 | 오늘 > 종료일 | 회색 |

### 인터랙션 설명 (목록)

| 요소 | 동작 | 결과 |
|------|------|------|
| [새 이벤트 등록] 클릭 | 등록 폼 표시 | 아래 이벤트 생성 폼으로 이동 |
| [수정] 클릭 | 편집 폼 표시 | 기존 데이터가 채워진 편집 폼 표시 |
| [삭제] 클릭 | 삭제 확인 팝업 | 확인 시 Event 삭제, 목록 갱신 |

## A04 이벤트 생성/편집 폼

### 와이어프레임

```
┌──────────────────────────────────────────────────────────────────────────┐
│  🐕 순찰대 관리자                                  관리자님 ▼   로그아웃  │
├────────────┬─────────────────────────────────────────────────────────────┤
│            │                                                            │
│   대시보드  │  ← 이벤트 목록으로 돌아가기                                 │
│   대원관리  │                                                            │
│   신고관리  │  이벤트 등록                                                │
│ > 이벤트   │  (편집 시: "이벤트 수정")                                    │
│   포인트   │                                                            │
│   Push    │  ┌─────────────────────────────────────────────────┐       │
│   통계    │  │                                                 │       │
│   콘텐츠   │  │  제목 *                                        │       │
│   설정    │  │  ┌─────────────────────────────────────────┐   │       │
│            │  │  │ 봄맞이 순찰 이벤트                       │   │       │
│            │  │  └─────────────────────────────────────────┘   │       │
│            │  │                                                 │       │
│            │  │  설명 *                                        │       │

│            │  │  │ 봄을 맞아 순찰 포인트를 1.5배로           │   │       │
│            │  │  │ 지급합니다! 많은 참여 부탁드립니다.       │   │       │

│            │  │  │                                         │   │       │
│            │  │  └─────────────────────────────────────────┘   │       │
│            │  │                                                 │       │
│            │  │  시작일 *                 종료일 *              │       │
│            │  │  ┌───────────────┐       ┌───────────────┐    │       │
│            │  │  │ 2026-03-01 📅 │       │ 2026-03-31 📅 │    │       │
│            │  │  └───────────────┘       └───────────────┘    │       │

│            │  │  └───────────────┘  예: 1.5 = 150% 포인트 지급 │       │
│            │  │                                                 │       │
│            │  │          [취소]                [저장]           │       │
```

### 인터랙션 설명 (폼)

| 요소 | 동작 | 결과 |
|------|------|------|
| 제목 입력 | 텍스트 입력 (필수) | 최대 100자 |
| 설명 입력 | 텍스트에어리어 입력 (필수) | 최대 1000자 |
| 시작일 | 데이트피커 선택 (필수) | 날짜 선택 |
| 종료일 | 데이트피커 선택 (필수) | 시작일 이후만 선택 가능 |
| 보너스 배율 | 숫자 입력 | 기본값 1.0, 소수점 1자리 (0.1 단위) |
| [취소] 클릭 | 작성 취소 | A04 이벤트 목록 복귀 |
| [저장] 클릭 | 유효성 검사 후 저장 | 아래 분기 참조 |

### 저장 결과 분기

| 조건 | 결과 |
|------|------|
| 유효성 통과 (신규) | Event 생성 (is_active=true) → "이벤트가 등록되었습니다" 토스트 → A04 목록 |
| 유효성 통과 (편집) | Event 업데이트 → "이벤트가 수정되었습니다" 토스트 → A04 목록 |
| 필수 항목 미입력 | 해당 필드 하단 에러: "필수 항목입니다" |
| 종료일 < 시작일 | 종료일 필드 에러: "종료일은 시작일 이후여야 합니다" |
