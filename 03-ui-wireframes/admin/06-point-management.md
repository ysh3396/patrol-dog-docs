## 인터랙션 설명

### 대상 검색

| 요소 | 동작 | 결과 |
|------|------|------|
| 검색 입력 | 2자 이상 입력 시 | 이름/이메일 자동완성 드롭다운 표시 |
| 자동완성 항목 클릭 | 대상 선택 | 선택된 대상 정보 카드 표시 (이름, 이메일, 현재 포인트) |

### 포인트 지급/차감

| 요소 | 동작 | 결과 |
|------|------|------|
| 유형 라디오 | "지급" 또는 "차감" 선택 | 지급: 포인트 추가, 차감: 포인트 감소 |
| 포인트 금액 | 양수 숫자 입력 (필수) | 0 이하 입력 불가 |
| 사유 입력 | 텍스트 입력 (필수) | 최대 200자 |
| [적용] 버튼 클릭 | 유효성 검사 후 확인 팝업 | 아래 분기 참조 |

### 적용 결과 분기

| 조건 | 결과 |
|------|------|
| 대상 미선택 | 에러: "대상을 선택해주세요" |
| 금액 미입력 | 에러: "금액을 입력해주세요" |
| 사유 미입력 | 에러: "사유를 입력해주세요" |
| 차감 시 현재 포인트 부족 | 경고 팝업: "현재 포인트(N pt)보다 큰 금액입니다. 계속하시겠습니까?" |
| 유효성 통과 | 확인 팝업 표시 |

### 적용 확인 팝업

```
┌──────────────────────────────────┐
│                                  │
│  포인트 지급 확인                 │
│                                  │
│  대상: 김순찰                     │
│  유형: 지급                       │
│  금액: +500 pt                   │
│  사유: 이벤트 보상                │
│                                  │
│  적용 후 포인트: 4,500 → 5,000pt │
│                                  │
│         [취소]     [확인]         │
│                                  │
└──────────────────────────────────┘
```

- 확인 시: User.points 변경 → "500 포인트가 지급되었습니다" 토스트 → 대상 정보 카드 갱신 → 이력 테이블 갱신
- 취소 시: 팝업 닫기, 폼 유지
