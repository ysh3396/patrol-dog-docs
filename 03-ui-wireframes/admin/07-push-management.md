### 인터랙션 설명 (이력)

|------|------|------|

## A06 Push 알림 발송 폼 (모달)

### 와이어프레임

```
┌──────────────────────────────────────────────────────────────────┐
│  새 알림 발송                                              [X]  │
│                                                                  │
│  대상 선택 *                                                     │
│  (●) 전체 (all)  — 모든 대원에게 발송                             │
│  ( ) 특정 그룹 (group)                                           │
│      ┌──────────────────────────────────────────────┐           │
│      │ □ 이번 주 순찰 미참여 대원                     │           │
│      │ □ 가입 후 첫 순찰 미완료 대원                  │           │
│      │ □ 포인트 1,000pt 이상 대원                    │           │
│      └──────────────────────────────────────────────┘           │
│      (그룹 선택 시 활성화)                                        │
│                                                                  │
│  알림 제목 *                                                     │
│  ┌──────────────────────────────────────────────────┐           │
│  │ 봄맞이 순찰 이벤트가 시작됩니다!                   │           │
│  └──────────────────────────────────────────────────┘           │
│                                                                  │
│  알림 내용 *                                                     │
│  ┌──────────────────────────────────────────────────┐           │
│  │ 3월 한 달간 순찰 포인트가 1.5배!                   │           │
│  │ 지금 바로 순찰을 시작해보세요.                     │           │
│  │                                                  │           │
│  └──────────────────────────────────────────────────┘           │
│                                                                  │
│  미리보기:                                                       │
│  ┌──────────────────────────────────────────────────┐           │
│  │  ┌──────────────────────────────────────────┐    │           │
│  │  │ 🐕 순찰대                                │    │           │
│  │  │ 봄맞이 순찰 이벤트가 시작됩니다!           │    │           │
│  │  │ 3월 한 달간 순찰 포인트가 1.5배!           │    │           │
│  │  │ 지금 바로 순찰을 시작해보세요.             │    │           │
│  │  └──────────────────────────────────────────┘    │           │
│  │              (모바일 알림 형태)                    │           │
│  └──────────────────────────────────────────────────┘           │
│                                                                  │
│  발송 대상: 1,234명                                              │
│                                                                  │
│                    [취소]              [발송]                     │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 인터랙션 설명 (발송 폼)

| 요소 | 동작 | 결과 |
|------|------|------|
| 대상 라디오: "전체(all)" | 선택 | 그룹 체크박스 비활성화, 발송 대상 수 = 전체 User 수 |
| 대상 라디오: "특정 그룹(group)" | 선택 | 그룹 체크박스 활성화 |
| 그룹 체크박스 | 1개 이상 선택 | 조건에 맞는 대상 수 실시간 표시 |
| 알림 제목 입력 | 텍스트 입력 (필수) | 최대 50자, 미리보기 실시간 반영 |
| 알림 내용 입력 | 텍스트 입력 (필수) | 최대 200자, 미리보기 실시간 반영 |
| 미리보기 영역 | 자동 업데이트 | 모바일 알림 형태로 제목/내용 표시 |
| [취소] 클릭 | 모달 닫기 | A06 이력 목록 복귀 |
| [발송] 클릭 | 유효성 검사 | 통과 시 발송 확인 팝업 |

### 발송 확인 팝업

```
┌──────────────────────────────────┐
│                                  │
│  알림 발송 확인                   │
│                                  │
│  1,234명에게 알림을               │
│  발송합니다.                      │
│                                  │
│  발송 후에는 취소할 수             │
│  없습니다.                       │
│                                  │
│         [취소]     [발송]         │
│                                  │
└──────────────────────────────────┘
```

### 발송 결과 분기

| 조건 | 결과 |
|------|------|
| 발송 성공 | PushNotification 생성 + FCM/APNs 전송 → "알림이 발송되었습니다" 토스트 → 모달 닫기 → 이력 목록 갱신 |
| 발송 실패 | 에러 메시지: "발송에 실패했습니다. 다시 시도해주세요." + [재시도] 버튼 |
| 필수 항목 미입력 | 해당 필드 하단 에러 표시 |
| 그룹 선택 시 대상 0명 | 경고: "선택한 조건에 해당하는 대원이 없습니다" |
