## 탭별 상세

### 탭 1: 포인트 정책

| 설정 항목 | 입력 유형 | 기본값 | 설명 |
|-----------|-----------|--------|------|
| 기본 포인트 지급률 | 숫자 입력 | 100 pt/km | 순찰 거리 1km당 지급 포인트 |
| 일일 포인트 상한 | 숫자 입력 | 1,000 pt | 하루 최대 적립 가능 포인트 |
| 이벤트 보너스 배율 | 숫자 입력 (소수) | 1.0x | 이벤트 기간 중 포인트 배율 |

### 탭 2: 어뷰징 방지

```
┌─────────────────────────────────────────┐
│  🚫 어뷰징 방지 설정                     │
│                                         │
│  속도 임계값 (초과 시 포인트 미지급)       │
│                                         │
│  도보 상한     [ 6   ] km/h             │
│  자전거 상한   [ 25  ] km/h             │
│  차량 감지     [ 40  ] km/h             │
│                                         │
│  연속 경고 횟수  [ 3 ] 회               │
│  ※ 초과 시 해당 순찰 포인트 전체 미지급   │
│                                         │
│  GPS 수집 간격                           │
│  ○ 거리 기반: [ 5 ] m 이동 시            │
│  ○ 시간 기반: [ 10 ] 초 간격             │
│  ● 혼합 (둘 중 먼저 충족 시)             │
│                                         │
│          [ 저장 ]  [ 초기화 ]             │
└─────────────────────────────────────────┘
```

| 설정 항목 | 입력 유형 | 기본값 | 설명 |
|-----------|-----------|--------|------|
| 도보 속도 상한 | 숫자 입력 | 6 km/h | 초과 시 비정상 이동으로 판단 |
| 자전거 속도 상한 | 숫자 입력 | 25 km/h | 자전거 허용 시 상한 |
| 차량 감지 속도 | 숫자 입력 | 40 km/h | 차량 이동 감지 기준 |
| 연속 경고 횟수 | 숫자 입력 | 3회 | 초과 시 해당 순찰 포인트 전체 미지급 |
| GPS 수집 간격 | 라디오 + 숫자 | 혼합(5m/10초) | 위치 수집 방식 |

## 인터랙션 설명

| 동작 | 결과 |
|------|------|
| [저장] 클릭 | 변경 사항 저장 + 성공 토스트 메시지 |
| [초기화] 클릭 | 기본값으로 리셋 확인 팝업 → 확인 시 초기화 |
| 탭 전환 | 저장하지 않은 변경 사항 있으면 경고 팝업 |
| 숫자 범위 초과 입력 | 유효성 검증 에러 메시지 (입력 필드 하단) |
| [비밀번호 변경] | 현재 비밀번호 + 새 비밀번호 + 확인 모달 |
| [카테고리 추가] | 인라인 입력 필드 추가 |
| [CSV 내보내기] | 기간 선택 팝업 → 다운로드 시작 |
