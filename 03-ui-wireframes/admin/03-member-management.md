### 인터랙션 설명

| 요소 | 동작 | 결과 |
|------|------|------|
| 검색바 입력 | 키워드 입력 후 Enter 또는 실시간 검색 | 이름/이메일 일치 대원 필터링 |
| 가입일 필터 | 날짜 범위 선택 | 해당 기간 가입 대원만 표시 |
| 활동상태 드롭다운 | [전체/활성/정지/탈퇴] 선택 | 해당 상태 대원만 표시 |
| 정렬 드롭다운 | [가입일순/활동량순/닉네임순] 선택 | 목록 재정렬 |
| [CSV 다운로드] 클릭 | 현재 필터 기준 CSV 생성 | 파일 다운로드 |
| [보기] 클릭 | 페이지 이동 | A02 대원 상세 화면 |
| 페이지네이션 번호 클릭 | 해당 페이지 이동 | 목록 갱신 (20건/페이지) |

### 인터랙션 설명

| 요소 | 동작 | 결과 |
|------|------|------|
| ← 대원 목록으로 돌아가기 | 클릭 | A02 대원 목록 화면 복귀 |
| 상태 드롭다운 | [활성/정지/탈퇴] 선택 | 상태값 변경 대기 |
| [저장] 버튼 클릭 | 상태 변경 API 호출 | 확인 팝업 표시 → 확인 시 저장, 성공 토스트 |
| 순찰 이력 행 클릭 | 순찰 상세 조회 | 해당 Patrol 상세 정보 모달 (경로 지도 포함) |
| 포인트 내역 페이지네이션 | 페이지 이동 | 내역 목록 갱신 |
