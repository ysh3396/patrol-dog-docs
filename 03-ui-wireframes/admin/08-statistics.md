## 레이아웃 설명

### 기간 선택 영역

| 요소 | 설명 |
|------|------|
| 프리셋 버튼 | [오늘] [이번주] [이번달] [전체] — 빠른 기간 선택 |
| 데이트 레인지 피커 | 시작일~종료일 직접 선택 — 프리셋보다 우선 적용 |

### 요약 카드 (4개)

| 카드 | 데이터 소스 |
|------|------------|
| 총 순찰 건수 | Patrol (status=completed) count in 선택 기간 |
| 총 이동 거리 | Patrol.distance_km SUM in 선택 기간 |
| 평균 순찰 시간 | Patrol.duration_seconds AVG in 선택 기간 (HH:MM:SS 포맷) |
| 총 신고 건수 | Report count in 선택 기간 |

### 차트 영역

| 차트 | 유형 | 데이터 |
|------|------|--------|
| 일별 순찰 추이 | 라인 차트 | 선택 기간 내 일별 Patrol(completed) 건수 |
| 시간대별 순찰 분포 | 세로 바 차트 | Patrol 시작 시간대(06~22시) 분포 |
| 카테고리별 신고 분포 | 파이 차트 | Report.category 비율 (불법투기/시설파손/안전위험/기타) |

### 지역별 히트맵

| 요소 | 설명 |
|------|------|
| 지도 | 선택 기간 내 Patrol Route 밀도를 히트맵으로 표시 |
| 확대/축소 | [+][-] 버튼 또는 스크롤 줌 |
| 범례 | 색상 농도별 순찰 건수 기준 |

## 인터랙션 설명

| 요소 | 동작 | 결과 |
|------|------|------|
| 프리셋 버튼 클릭 | 기간 자동 설정 | 모든 요약 카드 + 차트 + 히트맵 데이터 갱신 |
| 데이트 레인지 변경 | 시작일/종료일 선택 | 모든 데이터 갱신, 프리셋 선택 해제 |
| 차트 데이터 포인트 호버 | 마우스오버 | 툴팁: 해당 일/시간대의 수치 표시 |
| 히트맵 영역 클릭 | 특정 지역 선택 | 해당 지역 상세 수치 툴팁 |
| 순찰/대원/신고 체크박스 | 1개 이상 선택 | [CSV 내보내기] 버튼 활성화 |
| [CSV 내보내기] 클릭 | 선택한 데이터 CSV 생성 | 현재 기간 기준 CSV 파일 다운로드 |

### CSV 내보내기 데이터

| 데이터 유형 | 포함 필드 |
|------------|----------|
| 순찰 데이터 | 날짜, 대원명, 거리(km), 시간, 포인트, 마킹수, 상태 |
| 대원 데이터 | 이름, 이메일, 가입일, 총 순찰 수, 총 거리, 총 포인트 |
| 신고 데이터 | 날짜, 신고자, 카테고리, 위치, 상태, 처리일 |
