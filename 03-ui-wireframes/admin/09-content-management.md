## 영상 관리 탭

### 인터랙션 설명

| 요소 | 동작 | 결과 |
|------|------|------|
| [새 영상 등록] 클릭 | 등록 모달 열기 | 영상 등록 폼 모달 표시 |
| [수정] 클릭 | 편집 모달 열기 | 기존 데이터가 채워진 편집 모달 |
| [삭제] 클릭 | 삭제 확인 팝업 | "이 영상을 삭제하시겠습니까?" → 확인 시 삭제 |

### 영상 등록/수정 모달

```
┌──────────────────────────────────────────────┐
│  영상 등록                              [X]  │
│  (수정 시: "영상 수정")                       │
│                                              │
│  제목 *                                      │
│  ┌────────────────────────────────────────┐  │
│  │ 순찰 안전 가이드                        │  │
│  └────────────────────────────────────────┘  │
│                                              │
│  영상 URL *                                  │
│  ┌────────────────────────────────────────┐  │
│  │ https://youtu.be/abc123                │  │
│  └────────────────────────────────────────┘  │
│  YouTube 또는 Vimeo URL을 입력하세요          │
│                                              │
│  카테고리 *                                   │
│  (●) 교육    ( ) 홍보                        │
│                                              │
│              [취소]          [저장]           │
│                                              │
└──────────────────────────────────────────────┘
```

### 영상 폼 인터랙션

| 요소 | 동작 | 결과 |
|------|------|------|
| 제목 입력 | 텍스트 입력 (필수) | 최대 100자 |
| 영상 URL 입력 | URL 입력 (필수) | YouTube/Vimeo URL 형식 검증 |
| 카테고리 선택 | 교육/홍보 라디오 (필수) | 영상 분류 |
| [취소] 클릭 | 모달 닫기 | 목록 화면 복귀 |
| [저장] 클릭 | 유효성 검사 후 저장 | 성공: 토스트 + 목록 갱신, 실패: 에러 표시 |

### 게시판 관리 인터랙션

| 요소 | 동작 | 결과 |
|------|------|------|
| 카테고리 [수정] 클릭 | 수정 모달 | 카테고리명 변경 가능 (코드값은 변경 불가) |
| 신고된 게시글 [처리] 클릭 | 처리 모달 열기 | 아래 처리 모달 참조 |

### 신고 게시글 처리 모달

```
┌──────────────────────────────────────────────┐
│  신고 게시글 처리                        [X]  │
│                                              │
│  제목: 부적절한 광고 글                       │
│  작성자: 스패머1 (spammer@email.com)          │
│  카테고리: 자유 (free)                        │
│  신고 횟수: 5회                               │
│                                              │
│  신고 사유:                                   │
│  - 광고/스팸 (3건)                            │
│  - 부적절한 내용 (2건)                        │
│                                              │
│  게시글 내용 (미리보기):                       │
│  ┌────────────────────────────────────────┐  │
│  │ [게시글 본문 미리보기 영역]              │  │
│  └────────────────────────────────────────┘  │
│                                              │
│  처리 선택:                                   │
│  ( ) 유지 — 신고 기각                         │
│  (●) 숨김 — 게시글 비공개 처리                 │
│  ( ) 삭제 — 게시글 영구 삭제                   │
│                                              │
│              [취소]          [처리]           │
│                                              │
└──────────────────────────────────────────────┘
```

### 게시글 처리 분기

| 처리 유형 | 결과 |
|-----------|------|
| 유지 | 신고 건 기각 처리, 게시글 유지, 신고 횟수 초기화 |
| 숨김 | 게시글 비공개 처리 (작성자만 확인 가능), 작성자에게 알림 |
| 삭제 | 게시글 영구 삭제, 작성자에게 알림: "커뮤니티 정책 위반으로 게시글이 삭제되었습니다" |
