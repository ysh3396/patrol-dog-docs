**빈 상태 (산책 기록 없음)**:
```
┌─────────────────────────────┐
│  ┌─────────────────────┐    │
│  │ ○○의               🐾 │    │
│  │ 산책기록이에요.        │    │
│  │                     │    │
│  │  아직 산책 기록이 없어요  │    │ ← 빈 상태 안내
│  └─────────────────────┘    │
└─────────────────────────────┘
```

**동작 설명**:
- U02 순찰 대기 화면의 서브탭 "산책 일지" 탭 시 진입
- 반려견의 전체 누적 통계(총 횟수 / 총 거리 / 총 시간)를 상단 카드에 표시
- 상위 N% 뱃지: 전체 사용자 대비 해당 반려견의 산책 순위 백분위
- 월별 캘린더에서 산책한 날짜에 빨간 도트 표시
- 오늘 날짜는 font-bold + 원형 배경으로 강조

**인터랙션 노트**:
- 서브탭 "순찰하기" 탭 → [U02] (서브탭 전환)
- 날짜 셀 탭 → 해당 일의 순찰 기록 요약 (하단 슬라이드 카드 또는 [U08 순찰 기록 상세] 이동)
- 캘린더 헤더 "∨" 탭 → 월 선택 피커 표시
- 좌우 스와이프 → 이전/다음 월로 이동, 캘린더 헤더 월 갱신
- 캘린더 헤더 우측 수치 (0.0km | 0시간 0분) = 해당 월 합계
- 기록 없는 경우: 통계 카드 내 "아직 산책 기록이 없어요" 빈 상태 문구 표시
- 반려견 미등록 상태에서도 서브탭을 통해 진입 가능 (U25 빈 상태 표시)
