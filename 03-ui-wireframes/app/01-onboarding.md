# 01. 온보딩 와이어프레임 (U00 ~ U00d)

> **대상 화면**: U00 스플래시, U00a 온보딩 슬라이드, U00b 로그인, U00c 회원가입, U00d 비밀번호 찾기
> **참조**: `06-glossary.md`, `02-task-flows.md` 1장, `03-screen-transition.md` 1.1~1.2절

---

## U00 스플래시

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│                             │
│                             │
│                             │
│                             │
│         ┌─────────┐         │
│         │  🐕‍🦺    │         │ ← 앱 로고 (순찰대 심볼)
│         │ 순찰대앱  │         │
│         └─────────┘         │
│                             │
│                             │
│        ● ● ● 로딩 중        │ ← 로딩 인디케이터
│                             │
│                             │
│                             │
│                             │
│                             │
└─────────────────────────────┘
```

**동작 설명**:
- 앱 실행 시 로고와 로딩 인디케이터 표시 (약 2초)
- 로그인 토큰 유효 여부 확인
  - 토큰 유효 → [U02 순찰 대기] 자동 전환
  - 토큰 만료 + 최초 실행 → [U00a 온보딩 슬라이드]
  - 토큰 만료 + 재실행 → [U00b 로그인]

**인터랙션 노트**:
- 사용자 인터랙션 없음 (자동 전환)
- 하단 탭바 없음

## U00a 온보딩 슬라이드

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│                  건너뛰기 >  │ ← 우상단 건너뛰기 버튼
│                             │        탭 → [U00b]
│                             │
│    ┌───────────────────┐    │
│    │                   │    │
│    │   [일러스트레이션]   │    │ ← 슬라이드 이미지 영역
│    │                   │    │     (화면 50% 차지)
│    │   🐕 🚶 🗺️        │    │
│    │                   │    │
│    └───────────────────┘    │
│                             │
│    "우리 동네를 함께 지켜요"   │ ← 슬라이드 제목 (Bold)
│                             │
│    반려견과 함께 산책하며      │ ← 슬라이드 설명 (Gray)
│    안전한 동네를 만들어요      │
│                             │
│         ● ○ ○               │ ← 페이지 인디케이터
│                             │     (총 3장)
│                             │
│  ┌─────────────────────┐    │
│  │      시작하기         │    │ ← CTA 버튼 (마지막 슬라이드)
│  └─────────────────────┘    │     또는 "다음" (1~2번째)
│                             │     탭 → [U00b]
└─────────────────────────────┘
```

**슬라이드 내용**:

| 슬라이드 | 제목 | 설명 |
|---------|------|------|
| 1 | 우리 동네를 함께 지켜요 | 반려견과 함께 산책하며 안전한 동네를 만들어요 |
| 2 | 순찰하고 포인트 받으세요 | 걸은 거리만큼 포인트(Points)가 쌓여요 |
| 3 | 이상 상황을 바로 신고해요 | 현장 마킹(PatrolMark)과 신고(Report)로 동네를 안전하게 |

**인터랙션 노트**:
- 좌우 스와이프로 슬라이드 전환
- "건너뛰기" 탭 → [U00b 로그인]
- 마지막 슬라이드 "시작하기" 탭 → [U00b 로그인]
- 하단 탭바 없음

## U00b 로그인

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│                             │
│         ┌─────────┐         │
│         │  🐕‍🦺    │         │ ← 앱 로고 (소형)
│         │ 순찰대앱  │         │
│         └─────────┘         │
│                             │
│  ┌─────────────────────┐    │
│  │ 이메일               │    │ ← 이메일 입력 필드
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │ 비밀번호         👁️  │    │ ← 비밀번호 입력 필드
│  └─────────────────────┘    │     (비밀번호 표시/숨김 토글)
│                             │
│  ┌─────────────────────┐    │
│  │       로그인         │    │ ← 로그인 버튼 (Primary)
│  └─────────────────────┘    │     탭 → 인증 → [U02]
│                             │
│       비밀번호 찾기           │ ← 링크 텍스트
│                             │     탭 → [U00d]
│  ──────── 또는 ────────     │ ← 구분선
│                             │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐│
│  │카카오│ │네이버│ │ G  │ │ 🍎 ││ ← 소셜 로그인 4종
│  └────┘ └────┘ └────┘ └────┘│   카카오/네이버/Google/Apple
│                             │   탭 → 소셜 인증 웹뷰
│                             │   기존 User → [U02]
│  아직 회원이 아니신가요?       │   신규 User → [U00c]
│       회원가입               │ ← 링크 텍스트
│                             │     탭 → [U00c]
└─────────────────────────────┘
```

**에러 상태**:

| 에러 조건 | 메시지 | 위치 |
|----------|--------|------|
| 잘못된 비밀번호 | "비밀번호가 일치하지 않습니다" | 비밀번호 필드 하단 (빨간색) |
| 미가입 이메일 | "등록되지 않은 이메일입니다" | 이메일 필드 하단 (빨간색) |
| 소셜 인증 실패 | "인증에 실패했습니다. 다시 시도해주세요" | 토스트 메시지 |

**인터랙션 노트**:
- 이메일/비밀번호 입력 시 키보드 올라옴 → 화면 스크롤 대응
- 로그인 버튼은 이메일/비밀번호 모두 입력 시 활성화
- 로그인 성공 → Replace 전환으로 [U02 순찰 대기]
- 하단 탭바 없음

## U00c 회원가입

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 뒤로          회원가입    │ ← 네비게이션 바
│                             │     "← 뒤로" 탭 → [U00b]
├─────────────────────────────┤
│                             │
│      ┌──────────┐           │
│      │  👤 +    │           │ ← 프로필 사진 등록 (선택)
│      └──────────┘           │     탭 → 카메라/갤러리 선택
│                             │
│  이메일 *                    │
│  ┌─────────────────────┐    │
│  │ example@email.com    │    │ ← 실시간 중복 검사
│  └─────────────────────┘    │     ✓ 사용 가능 / ✗ 이미 사용 중
│                             │
│  비밀번호 *                  │
│  ┌─────────────────────┐    │
│  │ ••••••••        👁️  │    │ ← 8자 이상, 영문+숫자+특수문자
│  └─────────────────────┘    │
│  비밀번호 확인 *              │
│  ┌─────────────────────┐    │
│  │ ••••••••        👁️  │    │ ← 일치 여부 실시간 검사
│  └─────────────────────┘    │
│                             │
│  닉네임 *                    │
│  ┌─────────────────────┐    │
│  │ 순찰대원1             │    │ ← 2~10자, 실시간 중복 검사
│  └─────────────────────┘    │
│                             │
│  휴대폰 번호 *               │
│  ┌─────────────────────┐    │
│  │ 010-0000-0000        │    │ ← 형식 검사
│  └─────────────────────┘    │
│                             │
│  ☑ [필수] 이용약관 동의      │ ← 약관 동의 체크박스
│     보기 >                  │     "보기" 탭 → 약관 전문
│  ☑ [필수] 개인정보 처리방침   │
│     동의 보기 >              │
│  ☐ [선택] 마케팅 수신 동의   │
│                             │
│  ┌─────────────────────┐    │
│  │       가입하기        │    │ ← 가입 버튼 (필수 항목 완료 시 활성)
│  └─────────────────────┘    │     성공 → 토스트 + [U02]
│                             │
└─────────────────────────────┘
```

**유효성 검사**:

| 필드 | 조건 | 실시간 검사 |
|------|------|-----------|
| 이메일 | 이메일 형식 + 중복 검사 | O |
| 비밀번호 | 8자 이상, 영문+숫자+특수문자 | O |
| 비밀번호 확인 | 비밀번호와 일치 | O |
| 닉네임 | 2~10자, 중복 검사 | O |
| 휴대폰 번호 | 010-XXXX-XXXX 형식 | O |

**인터랙션 노트**:
- 스크롤 가능 (키보드 올라올 때 입력 필드 노출 보장)
- 필수 항목 미입력 시 가입 버튼 비활성(회색)
- 소셜 로그인 경유 시: 이메일 자동 채움(비활성), 닉네임/휴대폰만 입력
- 유효성 실패 → 해당 필드 하단 빨간색 에러 메시지
- 가입 성공 → "가입 완료" 토스트 → Replace 전환 [U02]
- 하단 탭바 없음

## U00d 비밀번호 찾기

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 뒤로      비밀번호 찾기   │ ← 네비게이션 바
│                             │     "← 뒤로" 탭 → [U00b]
├─────────────────────────────┤
│                             │
│                             │
│   가입 시 사용한 이메일을     │ ← 안내 텍스트
│   입력해주세요.              │
│   비밀번호 재설정 링크를      │
│   보내드립니다.              │
│                             │
│  이메일                      │
│  ┌─────────────────────┐    │
│  │ example@email.com    │    │ ← 이메일 입력 필드
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │  재설정 링크 발송      │    │ ← 발송 버튼
│  └─────────────────────┘    │
│                             │
│                             │
│                             │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─    │
│                             │
│  [발송 완료 상태]            │
│                             │
│  ✉️ 이메일이 발송되었습니다.  │ ← 발송 성공 메시지
│                             │
│  이메일에서 재설정 링크를     │
│  클릭해 새 비밀번호를         │
│  설정해주세요.               │
│                             │
│  메일이 오지 않나요?          │
│  [재발송]                    │ ← 재발송 링크
│                             │
│  ┌─────────────────────┐    │
│  │    로그인으로 돌아가기  │    │ ← 탭 → [U00b]
│  └─────────────────────┘    │
│                             │
└─────────────────────────────┘
```

**에러 상태**:

| 에러 조건 | 메시지 | 위치 |
|----------|--------|------|
| 미등록 이메일 | "등록되지 않은 이메일입니다" | 이메일 필드 하단 (빨간색) |
| 이메일 형식 오류 | "올바른 이메일 형식을 입력해주세요" | 이메일 필드 하단 (빨간색) |

**인터랙션 노트**:
- 발송 버튼은 이메일 입력 시 활성화
- 발송 성공 시 화면 하단에 완료 상태 표시 (화면 교체가 아닌 상태 전환)
- 재설정 링크 클릭 → 웹 비밀번호 변경 페이지 → 변경 완료 → 앱 [U00b]로 이동
- 미등록 이메일 시 "회원가입" 링크 제공 → [U00c]
- 하단 탭바 없음
