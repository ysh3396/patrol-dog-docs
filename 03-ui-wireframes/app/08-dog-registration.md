**동작 설명**:
- 진입 경로: [U02-empty] "반려견 등록하기" CTA, [U15] 반려견 관리 메뉴
- 신규 등록 모드: 모든 필드 빈 상태로 진입
- 편집 모드: 기존 반려견 정보가 pre-fill 된 상태로 진입
- 필수 필드(*) 모두 입력 시 하단 버튼 활성화
- 등록 완료 → "반려견이 등록되었습니다!" 토스트 → [U02]
- 수정 완료 → "반려견 정보가 수정되었습니다!" 토스트 → [U15]
- 삭제 → 확인 팝업 → 삭제 시 → [U02-empty]

**인터랙션 노트**:
- 필수 필드(*) 미입력 시 "반려견 등록하기" 버튼 비활성 (opacity 50%)
- 반려견 견종 필드 탭 → [U24a 견종 선택] 이동, 선택 완료 시 자동 복귀
- 성별 / 중성화 여부 / 예방접종 여부: 두 버튼 중 하나 선택 시 주황색 테두리로 표시
- 생년월일: 년도/월/일 드럼 롤러 피커, 미래 날짜 선택 불가
- 편집 모드: 하단에 "삭제하기" 텍스트 링크(빨간색) 표시
- 삭제 확인 팝업: "반려견 정보를 삭제하시겠습니까?" → [삭제] → [U02-empty], [취소] → U24 유지
- 프로필 사진 영역 탭 → 액션시트 (카메라 / 갤러리 / 기본이미지 사용)

- 견종 항목 탭 → 선택된 견종명을 U24 견종 필드에 입력 후 [U24] 복귀

- 검색 결과 없음 시: "일치하는 견종이 없습니다." 안내 텍스트 + "직접 입력" 옵션 유지
