## U02 순찰 대기 (홈)

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  🐕‍🦺 순찰대앱        🔔(3)  │ ← 헤더
│                             │     🔔 탭 → [U09 알림 목록]
├─────────────────────────────┤     배지 숫자 = 미읽음 알림 수
│  순찰하기   │   산책 일지     │ ← 서브탭 바 (현재: 순찰하기 활성)

│  │                     │    │
│  │                     │    │

│                             │
│  ┌─────────────────────┐    │
│  │ 최근 순찰             │    │ ← 최근 순찰 요약 카드
│  │ 2024.03.07  2.3km     │    │     날짜 | 거리
│  │ 45분  230pt           │    │     시간 | 획득 포인트(Points)

│        │  순찰 시작  │        │     (하단 중앙, 큰 원형)

└─────────────────────────────┘
```

**동작 설명**:
- 탭1(홈) 루트 화면
- 앱 진입 시 GPS 활성 상태 확인
- "순찰 시작" 탭 시:
  - GPS 권한 허용 → Patrol 레코드 생성(status=ready→active) → [U03]
  - GPS 권한 미요청 → [U21 위치 권한 요청 팝업] → 허용 시 [U03]
  - GPS 권한 거부/영구 거부 → 설정 안내 팝업
  - GPS 신호 없음 → "GPS 신호를 찾고 있습니다. 실외로 이동해주세요." 토스트
- 최근 순찰 기록 없으면 → "아직 순찰 기록이 없습니다" 안내 표시

**인터랙션 노트**:
- 지도 영역: 드래그(팬), 핀치 줌, 탭 가능
- 최근 순찰 카드 탭 → [U08 순찰 기록 상세]
- 순찰 진행 중(미완료 Patrol 감지) → 복구 팝업 표시
- 순찰 진행 중(U03/U04)에서는 서브탭 비표시

- 앱 백그라운드 전환 → GPS 수집 계속, 시스템 알림바 표시
- 앱 강제 종료 → 재실행 시 복구 팝업 제공

**인터랙션 노트**:
- U03과 동일 레이아웃, 상단 정보바의 시간/거리 카운터 멈춤
- 속도 0.0km/h로 표시
- 경과시간 옆에 ⏸️ 아이콘으로 일시정지 상태 시각적 표시

- 30분 초과 정지 시 자동 팝업
- 배터리 5% 이하 → 자동 일시정지 + 로컬 데이터 저장

## 지도 화면 인터랙션 스펙 테이블

> U02, U03, U04 공통 지도 인터랙션 정의

| 제스처/이벤트 | 동작 | 지도 반응 | 데이터 표시 |
|--------------|------|-----------|------------|
| 지도 탭 | 위치 확인 | 탭 지점에 마커 표시 | 주소 정보 툴팁 |
| 핀치 줌 인 | 확대 | 경로 디테일 확대, 마커 간격 넓어짐 | - |
| 핀치 줌 아웃 | 축소 | 경로 전체 보기, 마커 클러스터링 | - |
| 드래그(팬) | 지도 이동 | 뷰포트 변경 | - |
| 현위치 버튼 탭 | 현재 위치 센터링 | 현재 위치로 지도 이동 + 줌 복원 | GPS 좌표 |
| 마커(🔵/🔴) 탭 | 마킹/신고 상세 | 마커 하이라이트 + 말풍선 | 사진 썸네일 + 메모 미리보기 |
| 순찰 시작 | 트래킹 시작 | 현재 위치 센터링 + 경로(Route) 기록 시작 | 시간/거리/속도 카운터 시작 |
| 일시정지 | 트래킹 멈춤 | 경로 그리기 멈춤 + 정지 지점 ⏸️ 마커 | 카운터 멈춤, 속도 0 |
| 재개 | 트래킹 재시작 | 정지 구간 점선 연결 + 경로 그리기 재개 | 카운터 재시작 |
| 백그라운드 전환 | 앱 최소화 | 트래킹 계속 (Foreground Service) | 시스템 알림바에 거리/시간 표시 |
| 종료 | 트래킹 종료 | 전체 경로 표시 (줌 핏) | 최종 거리/시간/포인트(Points) |
| GPS 신호 유실 | 신호 없음 | 마지막 위치 유지 + 점선 구간 표시 | "GPS 신호 찾는 중" 배너 |
| 오프라인 전환 | 네트워크 끊김 | GPS 수집 계속, 경로 로컬 저장 | "오프라인 모드" 배너 |
