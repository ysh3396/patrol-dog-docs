├─────────────────────────────┤
│  ← 뒤로          설정       │ ← 네비게이션 바
│                             │     "← 뒤로" → [U15]
├─────────────────────────────┤
│                             │
│  알림 설정                   │ ← 섹션 헤더

**로그아웃 확인 팝업**:
```
┌───────────────────┐
│                   │
│  로그아웃          │
│                   │
│  로그아웃          │
│  하시겠습니까?     │
│                   │
│ [취소]  [로그아웃]  │
│                   │
└───────────────────┘
   ↓          ↓
 U20 유지   [U00b 로그인]
            (Replace 전환)
```

**회원 탈퇴 확인 팝업**:
```
┌───────────────────┐
│                   │
│  ⚠️ 회원 탈퇴      │
│                   │
│  탈퇴 시 모든      │
│  데이터가          │
│  삭제됩니다.       │
│                   │
│  순찰 기록, 포인트, │
│  게시글 등 모든    │
│  정보가 복구       │
│  불가능합니다.     │
│                   │
│ [취소]    [탈퇴]   │
│                   │
└───────────────────┘
   ↓          ↓
 U20 유지   탈퇴 처리
            → [U00b 로그인]
            (Replace 전환)
```

**위치 권한 상태 표시**:

| 권한 상태 | 표시 텍스트 | 색상 |
|----------|-----------|------|
| 항상 허용 | ✅ 항상 허용 | 초록 |
| 앱 사용 중 허용 | ⚠️ 앱 사용 중만 허용 | 주황 |
| 거부됨 | ❌ 거부됨 | 빨강 |
| 확인 안 됨 | ⚪ 미설정 | 회색 |

**인터랙션 노트**:
- 알림 토글: 각 항목별 ON/OFF 전환 (즉시 저장)
- 위치 권한 "설정" → OS 설정 앱으로 딥링크 이동
- 로그아웃 → 확인 팝업 → "로그아웃" → [U00b 로그인] Replace 전환
- 회원 탈퇴 → 확인 팝업 → "탈퇴" → User 삭제 처리 → [U00b 로그인] Replace 전환
- 이용약관/개인정보 처리방침 → 인앱 웹뷰로 전문 표시

**알림 유형별 아이콘 및 이동 화면**:

| 알림 유형 | 아이콘 | 탭 시 이동 화면 |
|----------|--------|--------------|
| 이벤트 알림 | 🎉 | [U02 홈] (이벤트 배너) |
| 신고(Report) 처리 알림 | 🚨 | [U08 순찰 기록 상세] |
| 커뮤니티 댓글/좋아요 | 💬 | [U11 게시글 상세] |
| 공지사항 | 📢 | [U10 피드] 또는 해당 화면 |
| 포인트 지급 알림 | 💎 | [U18 포인트 내역] |

**빈 상태** (알림 없음):
```
┌─────────────────────────────┐
│                             │
│         🔔                  │
│                             │
│   아직 알림이 없습니다.       │
│                             │
│                             │
└─────────────────────────────┘
```

**읽음/미읽음 시각 구분**:

| 상태 | 시각 표시 |
|------|----------|
| 미읽음 | 🔴 빨간 점 + 연한 파란 배경 |
| 읽음 | 빨간 점 없음 + 흰색 배경 |

**인터랙션 노트**:
- 모든 화면 헤더의 🔔 벨 아이콘에서 진입
- 벨 아이콘 배지 = 미읽음 알림 수
- 알림 카드 탭 → 읽음 처리 + 해당 화면으로 Push 전환 (딥링크)
- "전체 읽음" 탭 → 모든 미읽음 알림 읽음 처리 + 배지 초기화
- 알림 목록 무한 스크롤
- "← 뒤로" → 이전 화면으로 Pop 전환
- 당겨서 새로고침 지원
