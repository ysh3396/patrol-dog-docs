**인터랙션 노트**:
- 탭3(랭킹) 루트 화면
- 기간 탭 전환:
  - 실시간(daily): Patrol 종료(status=completed) 시 즉시 갱신
  - 주간(weekly): 월~일 누적, 매일 자정 갱신
  - 월간(monthly): 해당 월 누적, 매일 자정 갱신
- Top 3: 1위 중앙 배치, 2위 좌측, 3위 우측 (포디움 레이아웃)
- 순위 리스트: 무한 스크롤
- "내 순위" 바: 하단 고정, 강조 배경 색상
  - 탭 → 목록 내 내 순위 위치로 스크롤 이동
  - 내 순위가 Top 3에 포함되면 Top 3 영역에도 강조 표시
- 다른 User 항목 탭 → 동작 없음 (프로필 비공개)
- 당겨서 새로고침 (Pull-to-Refresh) 지원
