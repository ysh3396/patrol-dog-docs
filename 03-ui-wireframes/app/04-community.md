**인터랙션 노트**:
- 탭2(커뮤니티) 루트 화면
- 카테고리 탭 전환: 활동인증(activity) / 정보공유(info) / 자유(free)
- 게시글 카드 목록 = 무한 스크롤 (페이지네이션)
- 당겨서 새로고침 (Pull-to-Refresh)
- FAB(✏️) = 우하단 플로팅 버튼, 탭 → [U12]
- 영상 바로가기 → [U13]
- 순찰 진행 중일 때 상단 배너 "순찰 진행 중 | 탭하여 돌아가기" 표시

## U11 게시글 상세

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 뒤로     게시글 상세  ⋮  │ ← 네비게이션 바
│                             │     "← 뒤로" 탭 → [U10]
│                             │     ⋮ = 더보기 메뉴
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐    │
│  │ 👤 순찰대원1         │    │ ← 작성자 프로필
│  │    3분 전             │    │     프로필 이미지 + 닉네임
│  └─────────────────────┘    │     작성 시간
│                             │
│  오늘의 순찰 인증!            │ ← 제목 (Bold, 큰 글씨)
│                             │
│  ┌─────────────────────┐    │
│  │                     │    │
│  │    [이미지 1/3]       │    │ ← 이미지 영역
│  │                     │    │     좌우 스와이프 (최대 5장)
│  │                     │    │     페이지 인디케이터 ● ○ ○
│  └─────────────────────┘    │
│       ● ○ ○                 │
│                             │
│  오늘 강남구 역삼동 일대를    │ ← 본문 텍스트
│  순찰했습니다. 가로등이       │
│  고장난 곳을 발견하여         │
│  신고 접수했습니다.           │
│  다들 안전한 밤 보내세요!     │
│                             │
│  ┌─────────────────────┐    │
│  │  ❤️ 좋아요 12         │    │ ← 좋아요 버튼 + 카운트
│  └─────────────────────┘    │     탭 토글: 좋아요 추가/취소
│                             │     추가 시: likes_count+1
│                             │       + 하트 애니메이션
│  ─ ─ ─ 댓글 3개 ─ ─ ─      │ ← 댓글 섹션 구분선
│                             │
│  ┌─────────────────────┐    │
│  │ 👤 동네지킴이         │    │ ← 댓글 1
│  │ 수고하셨습니다! 👍     │    │     본인 댓글 → 롱프레스 → 삭제
│  │              2분 전  │    │
│  ├─────────────────────┤    │
│  │ 👤 산책러김씨         │    │ ← 댓글 2
│  │ 가로등 빨리 고쳐지면   │    │
│  │ 좋겠네요              │    │
│  │              5분 전  │    │
│  └─────────────────────┘    │
│                             │
│  ┌───────────────────┐ ┌─┐ │
│  │ 댓글을 입력하세요...  │ │▶│ │ ← 댓글 입력 바 (하단 고정)
│  └───────────────────┘ └─┘ │     ▶ = 등록 버튼
│                             │     빈 텍스트 시 비활성
├─────────────────────────────┤     최대 500자
│  🏠    💬    🏆    👤      │ ← 하단 탭바
│  홈   커뮤니티 랭킹  마이    │
└─────────────────────────────┘
```

**더보기(⋮) 메뉴**:

본인 게시글:
```
┌──────────────┐
│  수정          │ → [U12 편집 모드]
│  삭제          │ → 확인 팝업
│  취소          │     → 삭제 시 [U10]
└──────────────┘
```

타인 게시글:
```
┌──────────────┐
│  신고          │ → 신고 사유 선택
│  취소          │     → 제출 → 토스트
└──────────────┘
```

**인터랙션 노트**:
- 스크롤 가능 (긴 본문 + 댓글 목록)
- 이미지: 좌우 스와이프, 탭 시 전체화면 보기
- 좋아요: 탭 토글 (미좋아요→좋아요: likes_count+1, 하트 애니메이션 / 좋아요→취소: likes_count-1)
- 댓글 등록: 텍스트 입력 + ▶ 탭 → Comment 생성, 목록 갱신
- 본인 댓글 롱프레스 → "삭제" 확인 팝업 → Comment 삭제
- 키보드 올라올 때 댓글 입력 바 키보드 위로 이동

## U12 게시글 작성

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 취소     게시글 작성  등록 │ ← 네비게이션 바
│                             │     "← 취소" → 임시저장 팝업
│                             │       → [U10]
│                             │     "등록" → Post 생성
│                             │       → [U10]
├─────────────────────────────┤
│                             │
│  게시판 선택                 │
│  ┌────────┐ ┌────────┐      │
│  │활동인증 │ │정보공유 │      │ ← 게시판(board_type) 선택 칩
│  └────────┘ └────────┘      │     (택 1)
│  ┌────────┐                 │
│  │ 자유   │                 │
│  └────────┘                 │
│                             │
│  제목                       │
│  ┌─────────────────────┐    │
│  │ 제목을 입력하세요      │    │ ← 제목 입력 (필수, 최대 100자)
│  └─────────────────────┘    │
│                             │
│  본문                       │
│  ┌─────────────────────┐    │
│  │                     │    │ ← 본문 입력 (필수, 최대 2000자)
│  │ 내용을 입력하세요     │    │
│  │                     │    │
│  │                     │    │
│  │                     │    │
│  │                     │    │
│  │            0/2000   │    │ ← 글자 수 카운터
│  └─────────────────────┘    │
│                             │
│  이미지 첨부 (선택, 최대 5장)│
│  ┌────┐ ┌────┐ ┌────┐       │
│  │ 📷  │ │    │ │    │       │ ← 이미지 첨부 영역
│  │ +   │ │    │ │    │       │     탭 → 카메라/갤러리 선택
│  └────┘ └────┘ └────┘       │     권한 없을 시 → [U22]
│  ┌────┐ ┌────┐              │     첨부된 이미지: X 삭제
│  │    │ │    │              │
│  └────┘ └────┘              │
│                             │
├─────────────────────────────┤
│  🏠    💬    🏆    👤      │ ← 하단 탭바
│  홈   커뮤니티 랭킹  마이    │
└─────────────────────────────┘
```

**임시저장 팝업** ("← 취소" 또는 "뒤로" 시, 작성 내용 있을 때):
```
┌───────────────────┐
│                   │
│  작성 중인 내용이  │
│  있습니다.         │
│  나가시겠습니까?    │
│                   │
│ [계속 작성] [나가기]│
│                   │
└───────────────────┘
```

**편집 모드** (U11 → 수정 시):
- 기존 데이터 자동 채움 (제목, 본문, 이미지, board_type)
- "등록" 버튼 텍스트 → "저장"으로 변경
- 저장 성공 → "수정 완료" 토스트 → [U11 게시글 상세] (갱신된 내용)
- 변경 없이 저장 → "변경 사항이 없습니다" 토스트

**에러 상태**:

| 에러 조건 | 메시지 | 위치 |
|----------|--------|------|
| 게시판 미선택 | "게시판을 선택해주세요" | 게시판 영역 하단 (빨간색) |
| 제목 미입력 | "제목을 입력해주세요" | 제목 필드 하단 (빨간색) |
| 본문 미입력 | "내용을 입력해주세요" | 본문 필드 하단 (빨간색) |

**인터랙션 노트**:
- 스크롤 가능
- 필수 항목(게시판 + 제목 + 본문) 미입력 시 "등록" 버튼 비활성(회색)
- 등록 성공 → "게시글이 등록되었습니다" 토스트 → [U10 피드]
- 등록 실패(네트워크) → 에러 토스트 + 현재 화면 유지 (입력 데이터 보존)
