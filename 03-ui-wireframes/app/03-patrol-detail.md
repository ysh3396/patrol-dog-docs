## U05 현장 마킹 (바텀시트)

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│            순찰 진행 중       │ ← 헤더 (U03 위에 바텀시트)
├─────────────────────────────┤
│ 🕐 00:23:45 | 📏 1.2km     │ ← 상단 정보바 (타이머 계속 진행)
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │   [지도 영역 - 흐림]  │    │ ← 배경 지도 (반투명 오버레이)
│  │                     │    │
│  └─────────────────────┘    │
│                             │
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│ ← 바텀시트 시작 (드래그 핸들)
│  ── ──                      │
│                             │
│  현장 마킹                   │ ← 바텀시트 제목
│                             │
│  유형 선택                   │
│  ┌──────────┐ ┌──────────┐  │
│  │ 📌 일반   │ │ 🚨 신고   │  │ ← 유형 선택 칩 (택 1)
│  │   기록    │ │          │  │     "일반 기록" = PatrolMark
│  └──────────┘ └──────────┘  │       (type=normal)
│                             │     "신고" 선택 시 →
│  📍 서울시 강남구 역삼동 123  │       [U06 신고 작성]
│                             │ ← 현재 위치 자동 표시
│  사진 촬영                   │
│  ┌────┐ ┌────┐ ┌────┐       │
│  │ 📷  │ │    │ │    │       │ ← 사진 영역 (최대 3장, 선택)
│  │ +   │ │    │ │    │       │     탭 → 카메라 촬영 or
│  └────┘ └────┘ └────┘       │           갤러리 선택
│                             │     권한 없을 시 → [U22]
│  메모                       │
│  ┌─────────────────────┐    │
│  │ 특이사항을 입력하세요   │    │ ← 메모 입력 (최대 200자, 선택)
│  │ (최대 200자)          │    │
│  └─────────────────────┘    │
│                             │
│  ┌──────────┐ ┌──────────┐  │
│  │   취소    │ │   저장    │  │ ← 버튼
│  └──────────┘ └──────────┘  │     취소 → 바텀시트 닫힘 → [U03]
│                             │     저장 → PatrolMark 생성
└─────────────────────────────┘       → 지도에 🔵 핀 추가
                                      → 완료 토스트 → [U03]
```

**인터랙션 노트**:
- 바텀시트 형태로 U03 위에 오버레이 (타이머는 계속 진행)
- 유형 "신고" 선택 시 바텀시트 닫히고 → [U06 신고 작성] Push 전환
- 사진 촬영 시 카메라/갤러리 권한 미허용 → [U22 카메라/갤러리 권한 요청]
- GPS 좌표(lat, lng) 자동 기록
- 저장 성공 → 지도에 파란색(🔵) 핀 추가, "마킹이 저장되었습니다" 토스트
- 바텀시트 아래로 드래그 → 닫힘 (취소와 동일)

## U06 신고 작성

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 취소        신고 작성     │ ← 네비게이션 바
│                             │     "← 취소" 탭 → [U05] 복귀
├─────────────────────────────┤
│                             │
│  카테고리 (필수)             │
│  ┌──────┐ ┌──────┐          │
│  │불법투기│ │시설파손│          │ ← 카테고리 선택 칩
│  └──────┘ └──────┘          │     (택 1, 필수)
│  ┌──────┐ ┌──────┐          │
│  │안전위험│ │ 기타  │          │
│  └──────┘ └──────┘          │
│                             │
│  📍 서울시 강남구 역삼동 123  │ ← 현재 위치 자동 기록
│                             │
│  상세 설명 (필수)            │
│  ┌─────────────────────┐    │
│  │                     │    │ ← 텍스트 입력 영역
│  │ 상황을 상세하게       │    │     (최대 500자, 필수)
│  │ 설명해주세요.         │    │
│  │                     │    │
│  │                     │    │
│  │              0/500  │    │ ← 글자 수 카운터
│  └─────────────────────┘    │
│                             │
│  사진 첨부 (선택, 최대 3장)  │
│  ┌────┐ ┌────┐ ┌────┐       │
│  │ 📷  │ │    │ │    │       │ ← 사진 첨부 영역
│  │ +   │ │    │ │    │       │     탭 → 카메라/갤러리
│  └────┘ └────┘ └────┘       │     첨부된 사진: X 버튼으로 삭제
│                             │
│  ┌─────────────────────┐    │
│  │      신고 제출        │    │ ← 제출 버튼
│  └─────────────────────┘    │     (필수 항목 완료 시 활성)
│                             │
├─────────────────────────────┤
│  🏠    💬    🏆    👤      │ ← 하단 탭바
│  홈   커뮤니티 랭킹  마이    │
└─────────────────────────────┘
```

**동작 설명**:
- 제출 성공 시:
  - PatrolMark 생성 (type=report)
  - Report 생성 (status=submitted)
  - 지도에 빨간색(🔴) 핀 추가
  - "신고가 접수되었습니다" 토스트
  - → [U03 순찰 진행 중] 복귀 (타이머 연속)
- 네트워크 오류 시:
  - Report 데이터 로컬 저장 (오프라인 큐)
  - "네트워크 연결 시 자동으로 전송됩니다" 토스트
  - → [U03] 복귀

**에러 상태**:

| 에러 조건 | 메시지 | 위치 |
|----------|--------|------|
| 카테고리 미선택 | "카테고리를 선택해주세요" | 카테고리 영역 하단 (빨간색) |
| 상세 설명 미입력 | "상세 설명을 입력해주세요" | 설명 영역 하단 (빨간색) |

**인터랙션 노트**:
- 스크롤 가능 (키보드 올라올 때 대응)
- 필수 항목(카테고리 + 상세 설명) 미입력 시 제출 버튼 비활성(회색)
- "← 취소" 탭 → [U05 현장 마킹]으로 복귀

**저장 완료 후 선택**:
```
┌───────────────────┐
│                   │
│  순찰이 저장       │
│  되었습니다!       │
│                   │
│  💎 235pt 획득    │
│                   │
│ [홈으로]  [상세보기]│
│                   │
└───────────────────┘

```

**인터랙션 노트**:
- 스크롤 가능
- 포인트 획득 영역에 카운팅 애니메이션 효과
- 활동 일지(Patrol.note) 입력은 선택 사항
- "저장 및 완료" 탭 → Patrol 레코드 업데이트(note 저장), User 통계 증가, Ranking 갱신
- 저장 후 팝업에서 "홈으로" → [U02], "상세 보기" → [U08]

### U08 인터랙션 스펙 테이블

| 제스처/이벤트 | 동작 | 지도 반응 | 데이터 표시 |
|--------------|------|-----------|------------|
| 마커(🔵/🔴) 탭 | 마킹 상세 보기 | 해당 마커 하이라이트 (확대+강조 색상) | 사진+메모 바텀시트 |
| 경로 폴리라인 탭 | 구간 정보 보기 | 탭 구간 하이라이트 (굵기 변경) | 해당 구간 속도/시간 툴팁 |
| 핀치 줌 인 | 확대 | 경로+마커 디테일 확대 | - |
| 핀치 줌 아웃 | 축소 | 경로+마커 전체 보기, 마커 클러스터링 | - |
| 드래그(팬) | 지도 이동 | 뷰포트 변경 | - |
| 마킹 목록 항목 탭 | 해당 마킹 위치로 이동 | 지도 해당 마커 위치로 센터링+줌 | 마킹 상세 바텀시트 |
| 바텀시트 아래로 드래그 | 바텀시트 닫기 | 마커 하이라이트 해제 | - |
