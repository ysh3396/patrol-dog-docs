**인터랙션 노트**:
- 탭4(마이페이지) 루트 화면
- 프로필 이미지 탭 → 전체화면 이미지 보기
- 통계 카드: 총 순찰 횟수(total_patrols), 총 거리(total_distance_km), 총 포인트(points)
- 메뉴 항목 > 아이콘은 Push 전환 힌트
- 순찰 진행 중일 때 상단 배너 표시

**U16 vs U17 차이점**:

| 항목 | U16 활동 내역 | U17 순찰 기록 목록 |
|------|-------------|-----------------|
| 표시 형태 | 리스트 (텍스트 중심) | 카드 (미니 지도 포함) |
| 그룹핑 | 없음 (단순 시간순) | 월별 그룹 |
| 정보 밀도 | 높음 (마킹 수 포함) | 중간 (경로 시각화 중심) |
| 목적 | 활동 이력 빠른 조회 | 경로 시각적 비교 |

**인터랙션 노트**:
- 날짜별 그룹으로 순찰 기록 표시
- 미니 경로 지도: 축소된 경로(Route) 썸네일 (인터랙션 없음, 읽기 전용)
- 각 항목 탭 → [U08 순찰 기록 상세]
- 무한 스크롤

## U18 포인트 내역

```
┌─────────────────────────────┐
│  9:41          📶 🔋        │ ← 상태바
├─────────────────────────────┤
│  ← 뒤로       포인트 내역   │ ← 네비게이션 바
│                             │     "← 뒤로" → [U15]
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐    │
│  │                     │    │ ← 현재 보유 포인트(Points)
│  │   현재 보유 포인트     │    │     (큰 글씨, Bold)
│  │                     │    │
│  │     💎 8,530pt       │    │
│  │                     │    │
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │ 📅 2024.03.07        │    │ ← 포인트 이력 항목 1
│  │ 순찰 완료 (2.35km)   │    │     내용 설명
│  │              +235pt  │    │     적립(+) = 초록색
│  ├─────────────────────┤    │
│  │ 📅 2024.03.05        │    │ ← 포인트 이력 항목 2
│  │ 순찰 완료 (1.80km)   │    │
│  │              +180pt  │    │
│  ├─────────────────────┤    │
│  │ 📅 2024.03.03        │    │ ← 포인트 이력 항목 3
│  │ 이벤트 보너스 (x1.5)  │    │     보너스 배율 표시
│  │              +465pt  │    │
│  ├─────────────────────┤    │
│  │ 📅 2024.03.01        │    │ ← 포인트 이력 항목 4
│  │ 관리자 수동 차감      │    │     사용(-) = 빨간색
│  │               -50pt  │    │
│  ├─────────────────────┤    │
│  │ 📅 2024.02.28        │    │
│  │ 순찰 완료 (2.70km)   │    │
│  │              +270pt  │    │
│  ├─────────────────────┤    │
│  │ ...                  │    │ ← 무한 스크롤
│  └─────────────────────┘    │
│                             │
├─────────────────────────────┤
│  🏠    💬    🏆    👤      │ ← 하단 탭바
│  홈   커뮤니티 랭킹  마이    │
└─────────────────────────────┘
```

**포인트 이력 유형**:

| 유형 | 표시 내용 | 색상 |
|------|----------|------|
| 순찰 자동 지급 | "순찰 완료 (N.NNkm)" | +Npt (초록) |
| 이벤트 보너스 | "이벤트 보너스 (xN.N)" | +Npt (초록) |
| 관리자 수동 지급 | "관리자 수동 지급" + 사유 | +Npt (초록) |
| 관리자 수동 차감 | "관리자 수동 차감" + 사유 | -Npt (빨간) |

**인터랙션 노트**:
- 적립/사용 히스토리 리스트 (최신순, 무한 스크롤)
- 적립(+) = 초록색, 차감(-) = 빨간색으로 시각 구분
- 이력 항목 탭 → 동작 없음 (상세 정보가 이미 표시됨)
- 당겨서 새로고침 지원

**인터랙션 노트**:
- 프로필 사진: 탭 → 액션시트 (카메라 촬영 / 갤러리에서 선택 / 기본 이미지로 변경)
- 닉네임 변경 시 실시간 중복 검사
- 이메일은 표시만 (변경 불가, 회색 비활성)
- "저장" 탭 → 변경 사항 저장 → "프로필이 수정되었습니다" 토스트 → [U15]
- 변경 없이 "저장" → "변경 사항이 없습니다" 토스트
- "← 취소" → 변경 사항 있으면 확인 팝업: "변경 사항을 저장하지 않고 나가시겠습니까?"
